Claude Code Basics

A step-by-step guide to connecting Claude Code with GitHub

1
Verify

Check if Git is installed

Open a terminal (CMD, PowerShell, or Git Bash) and run:

git --version

You should see something like git version 2.53.0.windows.1. If not, download Git from git-scm.com.

2
Configure

Set your Git identity

Tell Git who you are. Claude ran these commands automatically:

git config --global user.name "Your Full Name"
git config --global user.email "your@email.com"

You can verify it worked with git config --global --list.

3
Authenticate

Connect to GitHub

The easiest way is the GitHub CLI. Install it from cli.github.com, then run:

gh auth login

Follow the prompts — it opens your browser to securely log in. You never need to type a password.

Alternatively, generate a Personal Access Token on GitHub (Settings → Developer Settings → PAT) and use it as your password when Git prompts you.

4
Connect

Clone or link a repository

To clone an existing repo from GitHub to your machine:

git clone https://github.com/<your-username>/<your-repo>.git

To link a new local folder to an existing GitHub repo:

git init
git remote add origin https://github.com/<your-username>/<your-repo>.git
5
Workflow

Push changes to GitHub

The standard workflow to save and upload your changes:

git add .
git commit -m "Your commit message"
git push origin main

Claude can run these commands for you — just ask!

6
Claude

Let Claude Code help you

Once Git is configured, Claude Code can:

• Read and edit files in your working directory

• Stage, commit, and push changes to GitHub

• Clone repos and create new projects

• Work with any repo you point it to

Your working directory is set to C:\Users\<your-username>\Documents\claude_projects.
