Open Claw

Open Source CLI TypeScript Advanced

Claude Code is open source. "Open Claw" is your guide to exploring the CLI repository — understanding how it works under the hood, running it locally from source, customising its behaviour, and building your own tools on top of it.

What you'll learn

  • Where to find the Claude Code open-source repository
  • How to clone, build, and run the CLI from source
  • The high-level architecture: tools, agents, permissions, and hooks
  • How to read and navigate a TypeScript monorepo of this scale
  • How to contribute patches, open issues, and engage with maintainers

Why explore the source?

  • Understand exactly what Claude can and cannot do with your filesystem
  • Learn how the tool-call loop and permission system are implemented
  • Build custom tool integrations that slot into the existing architecture
  • Debug unexpected behaviour by tracing it to the source
  • Contribute improvements and bug fixes back to the community

Architecture deep-dive

Try this prompt

"Read the Claude Code source and explain: (1) how new tools are registered, (2) how the permission system decides what to allow or deny, and (3) how hooks are triggered before and after tool calls. Show me the relevant files."

Adding a custom tool

Try this prompt

"Following the existing pattern in the Claude Code source, add a new tool called 'notify' that sends a desktop notification whenever Claude completes a long-running task. Show me where to register it and how to implement the handler."
