Todo App

JavaScript localStorage Beginner

The classic first JavaScript project. Build a task manager that lets you add, complete, and delete tasks — and remembers them after you close the browser.

What you'll learn

  • JavaScript DOM manipulation
  • Event listeners and handling user input
  • Storing and retrieving data with localStorage
  • Dynamically rendering a list from data

What to build

  • An input field to add new tasks
  • A list that displays all tasks
  • Checkboxes to mark tasks as complete
  • A delete button for each task
  • Persistence using localStorage so tasks survive page refresh

How to start with Claude Code

Try this prompt

"Build a todo app in a single HTML file. It should let me add tasks, mark them complete, and delete them. Save tasks in localStorage so they persist after refresh. Use a clean dark UI."
