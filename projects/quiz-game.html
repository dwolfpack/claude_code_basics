Quiz Game

JavaScript Logic Advanced Beginner

Build an interactive multiple-choice quiz with scoring, a timer, and a results screen. Learn how to manage game state and create a fun user experience.

What you'll learn

  • Managing application state (current question, score, timer)
  • Conditional rendering based on state
  • Using setInterval for countdown timers
  • Fetching questions from a public trivia API
  • Animating transitions between questions

What to build

  • A start screen with a category and difficulty selector
  • A question card with 4 multiple-choice answers
  • A countdown timer per question
  • Immediate feedback (correct/wrong highlight)
  • A final results screen with score and a "Play Again" button

How to start with Claude Code

Try this prompt

"Build a quiz game using the Open Trivia Database API. Show 10 multiple-choice questions one at a time with a 15-second timer per question. Highlight correct and wrong answers. Show the final score at the end with a play again button."
