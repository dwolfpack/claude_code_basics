Budget Tracker

JavaScript Charts Intermediate

Track income and expenses, see your balance at a glance, and visualise spending with charts. A practical app you'll actually use.

What you'll learn

  • Managing structured data (arrays of objects)
  • Calculating totals and balances in JavaScript
  • Using a charting library (Chart.js)
  • Building form-driven UIs
  • Persisting structured data with localStorage

What to build

  • A form to add income or expense entries with a label and amount
  • A running list of all transactions
  • A summary showing total income, total expenses, and balance
  • A pie or bar chart breaking down spending by category
  • The ability to delete entries and update totals live

How to start with Claude Code

Try this prompt

"Build a budget tracker app in a single HTML file using Chart.js. Let me add income and expense entries with a name, amount, and category. Show a live balance summary and a pie chart of spending by category. Save data to localStorage."
