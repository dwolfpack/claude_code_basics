Recipe Finder

API Search UI Intermediate

Search for recipes by ingredient or dish name using a free API. Build your first real search interface and learn to present data with images and cards.

What you'll learn

  • Building a live search interface
  • Fetching and displaying image-rich data from an API
  • Rendering dynamic card grids
  • Creating a detail view or modal for individual results
  • Filtering and sorting results

What to build

  • A search bar to find recipes by ingredient or name
  • A grid of recipe cards with images and titles
  • A detail page showing full ingredients and instructions
  • Filter by category (vegetarian, dessert, etc.)
  • A "save favourites" feature using localStorage

How to start with Claude Code

Try this prompt

"Build a recipe finder app using TheMealDB API. Let users search by ingredient or meal name. Show results as image cards. Clicking a card should show full recipe details including ingredients and steps."

Resources

TheMealDB API
Free recipe API with images, ingredients, and instructions. No key needed for basic use.
Spoonacular API
More advanced recipe API with dietary filters and nutrition info.
MDN – CSS Grid
Learn to build responsive card grids with CSS Grid layout.
javascript.info – DOM Searching
How to find and update elements when rendering search results.
