Personal Portfolio Website

HTML CSS Beginner

Build a personal website that showcases your skills, projects, and contact info. This is one of the most useful first projects — you'll use it forever.

What you'll learn

  • HTML structure and semantic elements
  • CSS layout with Flexbox and Grid
  • Responsive design for mobile and desktop
  • Deploying a static site with GitHub Pages

What to build

  • A hero section with your name and a short bio
  • A projects section with cards linking to your work
  • A skills section with icons or tags
  • A contact section with links to GitHub, LinkedIn, email

How to start with Claude Code

Try this prompt

"Create a personal portfolio website for me. Include a hero section, projects section, skills section, and contact links. Use a dark modern theme with HTML and CSS only."
