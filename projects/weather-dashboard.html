Weather Dashboard

API JSON fetch() Intermediate

Connect to a real weather API and display live data. Your first step into the world of APIs, async JavaScript, and data-driven UIs.

What you'll learn

  • Making HTTP requests with fetch()
  • Working with JSON data from an API
  • Async/await and handling Promises
  • Displaying dynamic data in the DOM
  • Handling API keys safely

What to build

  • A search bar to look up any city
  • Current temperature, weather condition, and icon
  • Humidity, wind speed, and feels-like temperature
  • A 5-day forecast section
  • Error handling for invalid cities

How to start with Claude Code

Try this prompt

"Build a weather dashboard using the OpenWeatherMap API. Let the user search by city and show current temperature, conditions, humidity, and wind. Include a 5-day forecast. Use a dark card-based UI."
